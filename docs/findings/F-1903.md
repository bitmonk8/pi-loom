## Finding

`H4b` and `H4c` each carry a determinism self-check as their first Tests bullet, asserting that replaying the same scripted inputs "yields the same observable transcript **on every run**". The gate behind that claim is a finite replay-and-compare: it executes a bounded number of replays of the scripted drive and compares their transcripts. "On every run" is a universal quantifier, but replay-and-compare can only witness determinism across the replays it actually executes — a non-determinism source that does not happen to diverge across those executed replays passes the gate green. The bullet therefore claims more than the mechanism can deliver.

Sibling gates in the same horizontal harness band scope their detection limit inline (e.g. `H5f`'s recogniser is described as a "best-effort inline-citation scan … not the assertion's semantic fidelity"). The two determinism bullets do not, so a reader takes the green as a universal determinism guarantee rather than as evidence over the executed replays.

## Solution Space

**Shape:** single

### Recommendation

In both leaves, scope the determinism claim to the replays the self-check executes. In `H4b-response-programming-surface.md` Tests bullet 1, replace the trailing clause "yields the same observable transcript on every run" with "yields an identical observable transcript across the self-check's replays". Apply the identical edit to `H4c-modeled-behaviour-surface.md` Tests bullet 1. The `Ships when.` lines in both leaves reference "the determinism gate"; leave those as-is — they already name the gate rather than restate the universal claim, so no edit is required there. The fix is one coherent wording change applied to the two leaves that share the bullet; the spec is read-only for this finding.

## Consequence

**Severity:** advisory

A reader (and the H6a release-gate auditor) treats the determinism self-check's green as a universal determinism guarantee, when it only witnesses determinism across the replays the self-check actually executes. A latent non-determinism source that does not diverge across those replays ships undetected behind a gate whose wording reads as exhaustive.
