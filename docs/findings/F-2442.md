## Problem

The *Slash-command entry* bullet under *Forwarding into `loomAbort`* in `cancellation.md` requires the runtime to "tolerate `ctx.signal` being `undefined` at slash-command entry … and MUST NOT depend on its truthiness for any pre-turn checkpoint." It pairs an input condition (`ctx.signal === undefined` at entry) with a prohibition but never states the positive observable behaviour that must result. The verb "tolerate" is undefined in the corpus, so a conformance author has no pass/fail oracle: nothing says whether dispatch proceeds, whether an exception or diagnostic is raised, or whether cancellation detection is deferred. The defect is a missing Then-clause on an otherwise normative MUST, not a contradiction with other text.

## Issue introduction

**Verdict:** single-commit
**Introducing commits:** 8395c03 — pi-loom spec: resolve "Cancellation source ctx.signal is undefined when a slash-handler starts" (2026-05-04, Thomas Andersen)
**History:** The slash-command entry bullet, including the "MUST tolerate `ctx.signal` being `undefined` … MUST NOT depend on its truthiness" clause, was authored in 8395c03 when the *Signal source* / *Forwarding into `loomAbort`* structure was introduced to resolve a prior finding about `ctx.signal` being undefined when a slash-handler starts. That fix established the input condition and the prohibition but never added the observable outcome, so the testability gap has been present since the clause first entered the corpus. The wording survived the later `docs/` relocation (31ff060) unchanged.

## Solution approach

Add a falsifiable observable outcome to the *Slash-command entry* bullet so the undefined-`ctx.signal`-at-entry rule names a pass/fail oracle. State that dispatch proceeds normally and that cancellation detection for this path is deferred to the first event-handler observation of an aborted `ctx.signal` (or an `agent_end` reporting a user-cancelled turn), which then fires `loomAbort.abort()` per the existing forwarding rule. Keep the existing "MUST NOT depend on truthiness" prohibition.
