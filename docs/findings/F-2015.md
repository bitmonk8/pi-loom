## Finding

A single callout block carries two anchor IDs using inconsistent version abbreviation conventions: `loom-1-0` (spells out "loom 1.0" with hyphens replacing the dot) and `v1` (abbreviated prefix). The rest of the corpus uses the `v1-seam-*` pattern for cross-references (confirmed by `tool-registration-lifetime.md`'s link `conversation-drive.md#v1-seam-mid-loom-user-session-replacement`), making `loom-1-0-seam-*` an outlier. Having both IDs is safe for backward compatibility but the naming inconsistency may mislead readers about which form is canonical.
