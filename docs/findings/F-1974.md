## Finding

The requirement is a scope constraint on the probe's implementation rather than an observable runtime outcome. No test predicate is specified. A conformant and a non-conformant implementation (one that adds a sixth check) are indistinguishable from the outside on any given host: on a conformant host both return no diagnostic; on a non-conformant host both return `loom/load/host-incompatible`. The only way to verify PIC-5 is code inspection, not a runtime assertion.
