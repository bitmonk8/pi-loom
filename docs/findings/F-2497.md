## Problem

`version-bump-intro.md`'s `#pi-version-bump-procedure` enumeration names "three 'requires re-validating' sentences elsewhere in the spec" and lists them by link. `discovery-sources.md`'s `#file-extension-namespace` section carries a fourth such sentence ŌĆö a Pi minor that adds a fourth `SlashCommandSource` arm (or `loomPaths` slot, or generic file-extension registry) MUST trigger a re-validation as part of the Pi version bump procedure. That fourth obligation is uncollected by the enumeration and the stated count is wrong.

## Issue introduction

**Verdict:** multi-commit-interaction
**Introducing commits:** f5e89f4 ŌĆö docs: enforce 100KB hard / 30KB recommended size cap on spec set (2026-06-04, Thomas Andersen); 11cabfe ŌĆö pi-loom spec-plan: fix F-0715 ŌĆö relocate three non-DI-seam blocks out of host-interfaces-services.md + repoint cross-refs (2026-06-22, Thomas Andersen)
**History:** Items (a), (b), (d), (e), and `version-bump-intro.md`'s "three requires-re-validating sentences" enumeration all entered in f5e89f4 (2026-06-04), the commit that split the monolithic spec into topic files ŌĆö the `# Queryerror` H1, the `### Query-time variants` grouping, the inline-only user-defined-error-types non-goal, and the `last_tool_name | null` comment were created in that form and never corrected. Item (f)'s mismatch is the interaction: the "three sentences" enumeration was authored in f5e89f4 to count three, and later 11cabfe (2026-06-22) added a fourth "MUST trigger a re-validation" sentence in `discovery-sources.md` without updating that count. The discovery and errors-and-results Non-goals sections (c, d) have been absent since the corpus's topic-split inception.

## Solution approach

Update `version-bump-intro.md`'s `#pi-version-bump-procedure` enumeration to include the `discovery-sources.md` `#file-extension-namespace` re-validation sentence: correct "three" to "four" and add a named cross-link to that paragraph alongside the three existing links, so every re-validation obligation resolves to the version-bump procedure steps from one place.
