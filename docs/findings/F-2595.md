## Problem

DISC-6 (`id="disc-6"`, `docs/spec_topics/discovery/package-and-settings.md`) pins the literal `.catch(() => {})` token as a normative MUST for silencing an abandoned per-read settlement, elevating an implementation detail to a conformance requirement. The observable contract is twofold and mechanism-independent: the abandoned read's eventual settlement must not re-enter the discovery pass and must not surface as an `unhandledRejection`; several constructs satisfy both. This contradicts the same paragraph, which delegates the equivalent await-construct choice to the implementer, naming `AbortController` only as an illustrative `e.g.`.

## Issue introduction

**Verdict:** single-commit
**Introducing commits:** `25e8a60` — "pi-loom spec: resolve \"T08 — Per-package.json read timeout: overrun is unbounded\"" (2026-05-07)
**History:** The corpus is git-tracked. `git log -S'.catch(() => {})' -- docs/` and `git show 25e8a60` show the per-read deadline mechanism — and with it the `.catch(() => {})` MUST — entered the normative spec in `25e8a60` (then in `discovery.md`, later relocated verbatim into the split topic file `discovery/package-and-settings.md` by `f5e89f4` without altering the token). That commit resolved the earlier "T08" finding; the fix's recommendation prose itself carried the line *"the runtime MUST attach a `.catch(() => {})` to silence unhandled-rejection warnings without re-routing the late result"*, and the mechanism token was copied verbatim into the normative body alongside the (correct) behavioural `MUST NOT be re-routed` clause. The over-prescription is therefore original to the per-read-deadline feature's introduction, not a later regression: there is no prior revision that stated the behavioural requirement mechanism-free.

## Solution approach

Rewrite the DISC-6 abandoned-read clause so that the two behavioural requirements — the settlement MUST NOT be re-routed into the discovery pass, and MUST NOT surface as an `unhandledRejection` — stay normative, and demote `.catch(() => {})` to a non-normative illustrative example. Match the paragraph's existing treatment of the await construct, which names `AbortController` only as an `e.g.`.
