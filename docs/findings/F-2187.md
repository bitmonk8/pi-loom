## Problem

Item 4 of **Package discovery → Roots scanned** directs the extension to read an "already-resolved" global npm root when `npmCommand` is configured, but names no source seam through which that resolved root reaches the extension. No `ExtensionContext` member (`host-interfaces-core.md`), no `FileSystem` member (PIC-13), and no `host-prerequisites.md` presupposition exposes a resolved global npm root. The pinned SDK `packages.md` §npm places global installs at `~/.pi/agent/npm/` and scopes `npmCommand` to the npm wrapper command, not the on-disk install root — so no documented Pi surface delivers a relocated resolved root, leaving an implementer with neither a seam to call nor a documented relocation to honour.

## Issue introduction
**Verdict:** single-commit
**Introducing commits:** 808c72b — pi-loom spec: resolve "npmCommand-resolved npm root presupposes a subprocess seam" (2026-06-06, Thomas Andersen)
**History:** The roots-scanned item entered the corpus at the spec-set split (f5e89f4, 2026-06-04) phrased as *"the extension uses the resolved global root reported by that command"*, which named its source — the `npmCommand` subprocess output. Commit 808c72b (2026-06-06) reworded item 4 to *"Pi resolves the global root and the extension reads that already-resolved root … the extension does not itself invoke `npmCommand`"* to close a subprocess-seam concern; in removing the subprocess it deleted the only named source without substituting a new seam, introducing the present unnamed-source gap.

## Solution approach

Rewrite item 4 to drop the `npmCommand` resolved-root clause and state that the extension scans the literal `~/.pi/agent/npm/` root, expanded through the `FileSystem.homedir()` seam per [DISC-1](./discovery-sources.md#disc-1). Record the future-relocation case — a custom `npmCommand` that relocates the global install root — as a deferred consideration that would require a named source (an `ExtensionContext`/`FileSystem` member or a `host-prerequisites.md` presupposition) rather than leaving the seam implicit.
