## Finding

`cancellation.md` carries seven cross-references whose link target is the bare hub file `./pi-integration-contract.md` with no fragment, each named after a specific PIC section the hub does not own (lines 5, 9, 15, 17, 19, 47, 49). Each named section lives on a sibling subpage in `pi-integration-contract/` (`subagent.md`, `conversation-drive.md`, `host-prerequisites.md`), each non-narrative under the `PIC` prefix and carrying coined `PIC-N` anchors (Subagent session lifecycle = `PIC-9`; Conversation-drive prompt-mode = `PIC-18`; Extension entry point obligation = `PIC-7`).

GOV-9 requires each cross-page reference to a normative rule on a non-narrative page to resolve as a `#prefix-n` fragment to the rule's anchor (a section-level link permitted only when the depended-upon page is narrative). The seven links target a narrative hub but name rules whose canonical home is a non-narrative subpage with an active `PIC-N` anchor — satisfying neither GOV-9 arm (not the rule's `#prefix-n`, not even the named heading, which is not on the hub).

The defect is a standing residue under GOV-9's progressive-drainage clause — the normalisation trigger has not fired on these links.

## Solution Space

**Shape:** single

### Recommendation

Repoint each of the seven fragmentless `./pi-integration-contract.md` links to the `#prefix-n` anchor on the subpage that owns the cited rule: Subagent session lifecycle refs (lines 5, 9b, 15, 47, 49) → `./pi-integration-contract/subagent.md#pic-9`; Conversation-drive prompt-mode ref (line 9a) → `./pi-integration-contract/conversation-drive.md#pic-18` (confirm during edit which obligation the sentence depends on); Extension entry point ref (line 19) → `./pi-integration-contract/host-prerequisites.md#pic-7` (or the section-level anchor if wayfinding only); bare ref (line 17, listener-cleanup) → `subagent.md#pic-9`. No new anchors needed — every cited obligation already carries a `PIC-N`. Leave link text unchanged. The already-anchor-targeted links elsewhere on the page are out of scope.

## Consequence

**Severity:** cosmetic

The cited subpages still hold the obligations and each `PIC-N` anchor is in place; a reader following a fragmentless hub link lands on an orientation page that links onward, so the cost is one extra hop. The defect is a standing GOV-9 conformance gap that falsifies the universal-resolvability contract — a tool grep-walking `#prefix-n` fragments will not see `cancellation.md`'s dependency on PIC-7/9/18.
