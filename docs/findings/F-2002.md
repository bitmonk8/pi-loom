## Finding

The slash-separated pair in dual-named PIC headings has inconsistent ordering. PIC-12 places the production name first (`Clock / FakeClock`); PIC-13 and PIC-14 place the fake name first (`FakeFileSystem / FileSystem`, `FakeFileWatcher / FileWatcher`). The HTML anchor IDs reflect the same inconsistency: `clock--fakeclock-interface`, `fakefilesystem--filesystem-interface`, `filewatcher-interface` (PIC-14 omits the fake name from the anchor entirely). There is no stated convention that explains which order to use.
