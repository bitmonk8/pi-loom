## Problem

The `#### System-prompt structure (normative)` sub-section of `docs/spec_topics/binder/binder-bypass-and-envelope.md` enumerates eight numbered structural obligations plus three reference-rendering tables and a description-omitted-form rule, each carrying RFC-2119 modal tokens, but none carries a `BNDR-N` REQ-ID anchor in the GOV-1 dual form. The page already uses `BNDR-1`…`BNDR-9` for other obligations, so each item here is addressable only by item number or by the heading slug `#system-prompt-structure-normative`. Without per-obligation anchors GOV-9's `#prefix-n` cross-link contract is unsatisfiable for any page that needs to depend on a single structural item, and downstream citations (the `V11d` / `V11d-T` plan leaves, coverage-matrix row `cka-45`) must enumerate the items as a block. The section is standing GOV-22 progressive-coinage residue.
