## Finding

over-prescribes implementation. Quote: "**Loom-level concurrency primitives** (e.g. `parallel { ... }` blocks or a parallel-`for` form) **building on Pi tools' Promise-returning shape**". The deferred capability is "loom-level concurrency"; the clause "building on Pi tools' Promise-returning shape" binds a not-yet-designed feature to a particular implementation substrate without justification. The surface syntax examples (`parallel { ... }`) are appropriate to convey scope, but the substrate constraint is an implementation choice that a future design need not honour.
