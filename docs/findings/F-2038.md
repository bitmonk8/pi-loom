## Finding

The sentence "No Pi-side change is required for loom 1.0; if Pi later adds `pi.unregisterTool`, the registration cache becomes the natural site to call it from on extension teardown without altering the per-mode wiring rule above" is a future-phase design note embedded in normative prose without any label or separation. It is not a loom 1.0 requirement; it is a forward design sketch. Left unlabeled and unseparated from normative text, it can be mistaken as an implementer obligation or a loom 1.0 invariant, when it is neither.
