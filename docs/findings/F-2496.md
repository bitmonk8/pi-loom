## Problem

The `| null` branch on `ToolLoopExhaustedError.last_tool_name` in `queryerror-variants.md` has no loom 1.0-reachable case and is recorded only as an inline schema comment. The page's comparable forward-compat openness — the discriminator type-openness — is instead formalised as a `loom 1.0 seam` blockquote anchored at `ERR-15`. The asymmetry leaves a deferred-surface boundary documented informally, which obscures whether the branch is intentionally reserved or an oversight.

**Verdict:** multi-commit-interaction
**Introducing commits:** f5e89f4 — docs: enforce 100KB hard / 30KB recommended size cap on spec set (2026-06-04, Thomas Andersen); 11cabfe — pi-loom spec-plan: fix F-0715 — relocate three non-DI-seam blocks out of host-interfaces-services.md + repoint cross-refs (2026-06-22, Thomas Andersen)
**History:** Items (a), (b), (d), (e), and `version-bump-intro.md`'s "three requires-re-validating sentences" enumeration all entered in f5e89f4 (2026-06-04), the commit that split the monolithic spec into topic files — the `# Queryerror` H1, the `### Query-time variants` grouping, the inline-only user-defined-error-types non-goal, and the `last_tool_name | null` comment were created in that form and never corrected. Item (f)'s mismatch is the interaction: the "three sentences" enumeration was authored in f5e89f4 to count three, and later 11cabfe (2026-06-22) added a fourth "MUST trigger a re-validation" sentence in `discovery-sources.md` without updating that count. The discovery and errors-and-results Non-goals sections (c, d) have been absent since the corpus's topic-split inception.

## Solution approach

Pick one direction and stop leaving the branch as a bare inline comment. Either promote the `| null` branch's forward-compat rationale into a `loom 1.0 seam` blockquote on `queryerror-variants.md`, anchored with a REQ-ID under the page's `ERR` prefix and matching the existing `ERR-15` seam style; or narrow `last_tool_name` to `string` and delete the `| null` branch if the future case is not worth reserving.
