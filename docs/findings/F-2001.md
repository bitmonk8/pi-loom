## Finding

A filesystem write failure (e.g. `EACCES`, `EPERM`, `ENOSPC`, read-only volume) when the runtime performs "every other normative file read **or write** the runtime performs." - **What is missing:** Rejection-mapping and surfacing for `writeText`. Every read-side member of the seam documents its Node-style `.code` rejection contract verbatim Рђћ `readText` (`ENOENT`/`EACCES`/`EPERM`/other), `readBytes`, `readdir`, `lstat`, `realpath` (`ELOOP`/`ENOENT`/Рђд).
