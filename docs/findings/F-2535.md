## Problem

The "Automatic context escalation" item under the "Surface extensions gated on an open design decision" sub-section (`id="surface-extensions-open-design"`) in `surface-extensions.md` carries a `*Decision required before this item can be scoped:*` line — an unresolved open design TBD living inside the normative spec corpus. Open decisions belong in an issue tracker or ADR, not parked in the spec.

## Issue introduction

**Verdict:** multi-commit-interaction
**Introducing commits:** 0e12140 — pi-loom spec: resolve "Non-goals are not consolidated into a single section" (2026-05-08, Thomas Andersen); 02d7ca5 — pi-loom spec: resolve "T02 surface-extension seam obligations not consistently marked" (2026-05-08, Thomas Andersen); 528cef4 — pi-loom spec: resolve "Surface extensions → Automatic context escalation — embedded open decision" (2026-06-03, Thomas Andersen)
**History:** The three cruft sites entered separately. The concurrent-sessions detection-gap prose was seeded when the non-goals were consolidated (0e12140, 2026-05-08) and then accreted across later edits (e.g. the build-time SDK surface-inventory assertion detail arrived with 0229468, 2026-06-06). The "not counted in the tally" exclusion preamble was added with the seam-marking pass (02d7ca5, 2026-05-08). The embedded open-decision item was introduced when the Automatic-context-escalation decision was moved into the surface-extensions page (528cef4, 2026-06-03). The 81bf52a collapse of `spec/` into `spec_topics/` (2026-06-11) relocated all three but did not introduce them.

## Solution approach

Move the "Automatic context escalation" item and its `*Decision required*` line out of `surface-extensions.md` to an issue/ADR stub outside the normative corpus. Optionally leave a forward pointer (item name + link) under the existing defensive seam callout at `binder/binder-model-and-context.md#v1-seam-automatic-context-escalation`, which already pins the loom 1.0 runtime invariant. Delete the now-empty "gated on an open design decision" sub-section heading (`id="surface-extensions-open-design"`) if no item remains.
