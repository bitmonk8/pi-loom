## Finding

The MUST enumerates four permitted trajectory labels but provides no acceptance criterion for when an annotation is *adequate* — i.e., how specifically it must describe which of the four trajectories applies and what inputs reach that trajectory. The annotation could trivially satisfy the letter of the MUST by adding "succeeds" to every retired ceiling's page without any supporting rationale, because the spec does not require the annotation to be *correct* or *complete* for the actual post-retirement runtime behaviour on the relevant input class.
