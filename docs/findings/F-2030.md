## Finding

"The minimal edit required" has no acceptance criterion. To falsify minimality, a reviewer must demonstrate that a strictly smaller change would still keep the shared structure well-formed — but "well-formed" for a shared message-template row, header line, or discriminator enumeration is never defined for the attribution test. Without a definition of well-formedness, there is no oracle against which to measure whether a proposed edit is minimal or whether a smaller subset would also have been sufficient. A reviewer cannot issue a reproducible pass/fail on condition (iii).
