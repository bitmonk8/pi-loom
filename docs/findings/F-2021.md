## Finding

Group A of the always-log set (enumerated two paragraphs above) explicitly includes "Binder failures (every row of [Binder — Failure modes]…)" — these are not `QueryError` kinds. The explanatory sentence incorrectly describes all group-A members as "`QueryError` kinds," which understates group A's scope. An implementer reading this passage to understand the always-log set composition may conclude that only `QueryError`-kinded failures qualify for group A and fail to implement the binder-failure rows.
