## Problem

`V4c` defers the live-surface confirmation of `ERR-8` to `V9i`, and `coverage-matrix.md`'s `ERR-8 … ERR-12` row names `V9i` as the live subagent-surface ERR-8 co-witness, so `V9i` is the authoritative closing leaf for the live-surface arm of ERR-8. That obligation appears only in the `V9i` implementation leaf's `Tests` list (the delegated live-carrier witness bullet); the paired `V9i-T` tests leaf carries no ERR-8 bullet, and `V9i`'s `Ships when` names only `dispose()`-in-`finally` idempotency. Under the per-phase TDD ritual the failing assertion for each REQ-ID is authored in the `-T` leaf and gated by `Ships when`, so the live-surface ERR-8 witness is never authored red and `V9i-complete` can be tagged while the cancellation-non-mutation behaviour on the real subagent `AgentSession` goes unverified anywhere in the corpus.
