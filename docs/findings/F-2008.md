## Finding

The edge cases section of the same file explicitly mandates boundary-condition test vectors at the category-2 80/81 code-point boundary and calls out the category-5 identifier-shape predicate, but provides no equivalent boundary or escape test vectors for the `<failure>` discriminator's 256-UTF-16-code-unit truncation rule or its control-character escape rule. The rules are stated normatively, but without vectors at the 256/257-code-unit boundary and the no-escape / needs-escape character boundary, a conformance test cannot declare a mechanically checkable pass/fail for those steps.
