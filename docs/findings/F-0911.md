## Finding

NOCEIL labels are a categorically distinct identifier class from numbered REQ-IDs; a reviewer needing to distinguish executable REQ-IDs (which drive the H5a citing-test obligation) from GOV-16 inline labels cannot do so from the current summary. (Note: this concerns the review-machinery projection index `index/ids.md`, not a plan-corpus defect — likely out-of-scope at triage.)

**Suggested fix:** Either rename the header to "Referenced spec identifiers (REQ-IDs and GOV-16 inline labels)", or split into "Executable REQ-IDs" (NOCEIL removed) and "GOV-16 inline labels" (NOCEIL + HC3).
